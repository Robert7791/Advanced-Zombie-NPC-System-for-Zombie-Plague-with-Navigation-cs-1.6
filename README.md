Требования:
•	Zombie Plague (модификация)
•	ReAPI

Мультиязычность:

1.	RU

Представляю плагин для AMX Mod X, который добавляет в режим Zombie Plague умных NPC-зомби с продвинутой системой навигации.

  Основные возможности:

1. Плагин boss_zombiesystem.sma:


•	7 уникальных типов NPC-зомби: Классик, Деймос, Призрак, Лекарь, Ведьма, Большой и Гончая.
•	Индивидуальный ИИ: У каждого зомби свои навыки, атаки и поведение (лечение, невидимость, гранаты и др.).
•	Оптимизация производительности:
o	Кеширование путей для снижения нагрузки на CPU.
o	Адаптивная частота обновления think в зависимости от дистанции до игрока.
o	Плавное движение и повороты без рывков.
•	Избегание коллизий: NPC не сталкиваются друг с другом и с зомби-игроками.
•	Поддержка навигационной системы: Интеграция с NavigationSystem для поиска пути.
•	Гибкие настройки: Лимит NPC, урон, здоровье, скорость и звуки.
2. Плагин NavigationSystem.sma:


•	Автоматическое построение навигационной сетки: Генерация нодов и путей на карте.
•	Поиск пути A*: Эффективный алгоритм для NPC.
•	Визуализация: Режим редактирования для отладки и настройки путей.
•	Поддержка лестниц, дверей и разрушаемых объектов.
•	Сохранение и загрузка навигационных данных.

3. Плагин Npc_example_spawn.sma:

Пример создания точки спавна по команде в чате /set
